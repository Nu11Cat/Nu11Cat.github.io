```java
//源码：
public static int binarySearch(int[] a, int key) {
    int low = 0, high = a.length - 1;
    while (low <= high) {
        int mid = (low + high) >>> 1;
        int midVal = a[mid];

        if (midVal < key)
            low = mid + 1;
        else if (midVal > key)
            high = mid - 1;
        else
            return mid; // 找到了
    }
    return -(low + 1); // 没找到，返回插入点
}
```

#### 模板：

左闭右闭： while(<=)   right = mid - 1;

左闭右开：while(<)  right = mid;
