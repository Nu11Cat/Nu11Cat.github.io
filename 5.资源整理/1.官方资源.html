1.官方资源

5/24/25About 3 min

1. Java 官方资源

  • Java 官网 & JDK 下载：获取最新版本的 JDK（Java Development Kit）和相关安装包，支持 Windows、macOS 和 Linux 系统。
  • JDK 官方文档：包括 Java 语言、API 和工具的官方文档，详细解释了每个 Java 版本的特性和规范。
  • JVM 官方文档：深入了解 Java 虚拟机的内部机制、JVM 规范和性能调优的最佳实践。
  • Java SE 技术文档：包括 Java SE（Standard Edition）的各种技术文档，包括库、工具、性能调优和安全性等内容。

2. Spring 官方资源

  • Spring 官网：Spring Framework 的官方首页，包含 Spring Boot、Spring Cloud、Spring Security、Spring Batch 等技术的官方文档、教程和示例项目。
  • Spring 官方文档：提供 Spring 框架的详细文档和 API 参考，帮助开发者快速上手和深入了解 Spring 技术栈。
  • Spring Boot 官方文档：Spring Boot 的官方文档，介绍如何快速构建和部署基于 Spring 的应用程序。
  • Spring Cloud 官方文档：针对微服务架构的 Spring Cloud 框架的详细官方文档，介绍了微服务开发、服务发现、负载均衡等内容。

3. MyBatis 官方资源

  • MyBatis 官网：MyBatis 框架的官方页面，提供文档、示例和插件。
  • MyBatis 官方文档：包括 MyBatis 的入门教程、核心概念和高级功能，适合 Java 开发者快速上手。
  • MyBatis-Spring 官方文档：讲解如何在 Spring 环境下高效集成 MyBatis。

4. Hibernate 官方资源

  • Hibernate 官网：Hibernate ORM 框架的官网，提供官方文档、教程、最佳实践和示例代码。
  • Hibernate ORM 官方文档：深入讲解 Hibernate ORM 的配置、性能优化和使用方式。

5. Maven 官方资源

  • Maven 官网：提供 Maven 构建工具的官方下载与入口文档。
  • Maven 官方文档：涵盖从基础命令到高级配置的完整文档合集。

6. Gradle 官方资源

  • Gradle 官网：Gradle 是现代化的构建工具，支持 Java 项目构建、发布和自动化部署。
  • Gradle 官方文档：包含使用教程和 API 参考，详细介绍构建逻辑和脚本。

7. Docker 官方资源

  • Docker 官网：Docker 平台的门户，提供容器技术的学习资料和工具下载。
  • Docker 官方文档：包括基础知识、命令行使用、Dockerfile 编写和 Java 项目容器化实践。

8. Kubernetes 官方资源

  • Kubernetes 官网：Kubernetes 是主流容器编排平台，官网提供部署、监控、服务治理等资料。
  • Kubernetes 官方文档：包括安装、API 使用、YAML 配置、与 Spring Boot 等集成等内容。

9. Nginx 官方资源

  • Nginx 官网：提供商业版和开源版 Nginx 的下载与概览。
  • Nginx 官方文档：包括反向代理、负载均衡、安全配置、缓存优化等实战指南。

10. Git 官方资源

  • Git 官网：Git 版本控制工具的官网，提供下载和基础介绍。
  • Git 官方文档：涵盖 Git 的所有核心命令、操作原理与分支管理机制。
